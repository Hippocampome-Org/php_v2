How to use the csv output to create SynproEvidenceFragmentRel and SynproEvidencePropertyTypeRel tables:

The files evi_pro_type_rel_nbyk.csv and evi_pro_type_rel_nbym.csv in csv2db are where results from
running the data processing here are be stored. These csv files will create 
SynproEvidenceFragmentRel and SynproEvidencePropertyTypeRel when building the hippocampome database
is done. Place the files generated in the iconv folder in the csv2db software then rebuild the
hippocampome database with that software.
